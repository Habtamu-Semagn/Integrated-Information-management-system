# Product Overview

## System Name
Startup Support System

## Purpose
A web platform for registering startups and enabling government staff to manage, review, and support them.

## Core Users

### 1. Applicant (Startup Owner)
- Register/login
- Submit startup application
- Track status

### 2. Government Staff
- Login securely
- View applications
- Review and update status

### 3. Admin (Optional)
- Manage users
- Manage system settings

## Core Features (Phase 1)

- Authentication (JWT)
- Submit startup application
- View applications (staff)
- Update application status

## Future Services (Phase 2+)

- Funding service
- Mentorship service
- Training programs
- Reporting & analytics
- Notifications system

## Goal

Build a scalable system that can support multiple services in a single platform without becoming messy.
